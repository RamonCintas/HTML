Criar Uma Página Web Utilizando as Tags Aprendidas

Entendendo o Desafio: A ideia deste Desafio de Projeto é elaborar um website, utilizando uma estrutura bem básica, utilizando tags aprendidas durante as aulas práticas:

Índice

INSTRUÇÕES: Utilizar todas as tags explicadas nas aulas: h1 até h6, p, mark, small, i, u, strong, ol, ul, li, a, hr, sub, sup, blockquote; Utilizar novas tags sugeridas: font, del, abbr (a ideia é buscar estas tags na internet, entender como ela funciona e utilizar no texto).

Observação 1: Coloque com suas palavras o que entendeu sobre o assunto. Não esqueça de colocar links para outros sites e também de formatar o texto com negrito, itálico e sublinhado quando necessário.

Não esqueça também de colocar o link para voltar até o índice. (Lembra das âncoras?)

Observação 2: Coloque links para outras páginas que falam sobre estes assuntos. Não esquece de que precisa abrir em outra aba/janela do navegador.

h1 até h6

(Voltar para o Índice)

(Heading 1 até Heading 6):Essas tags são usadas para criar títulos e subtítulos em uma página HTML. O h1 é o título mais importante e o h6 é o menos importante. A aparência e o tamanho do texto podem variar dependendo do navegador e do estilo aplicado.

p

(Voltar para o Índice)

(Paragraph):Essa tag é usada para criar parágrafos de texto. Ela envolve um bloco de texto e o navegador exibe uma quebra de linha antes e depois do texto do parágrafo.

mark

(Voltar para o Índice)

(Marked Text):Essa tag é usada para destacar ou marcar um trecho de texto. O navegador geralmente exibe o texto marcado com uma cor de fundo amarela ou outra cor que indique destaque.

small

(Voltar para o Índice)

(Small Text):Essa tag é usada para definir um texto com uma fonte menor do que o tamanho de texto padrão. É útil para notas de rodapé, legendas ou qualquer outro conteúdo secundário que seja menos importante.

i

(Voltar para o Índice)

(Italic):Essa tag é usada para aplicar uma forma de texto em itálico. O texto dentro dessa tag será exibido inclinado.

u

(Voltar para o Índice)

(Underline):Essa tag é usada para aplicar uma linha de sublinhado ao texto. O texto dentro dessa tag será exibido com uma linha horizontal abaixo dele.

strong

(Voltar para o Índice)

(Strong Importance):Essa tag é usada para dar ênfase ou importância extra a um trecho de texto. O texto dentro dessa tag geralmente é exibido em negrito.

ol

(Voltar para o Índice)

(Ordered List):Essa tag é usada para criar uma lista ordenada, na qual cada item da lista é numerado. Os itens da lista são definidos pela tag li.

ul

(Voltar para o Índice)

(Unordered List):Essa tag é usada para criar uma lista não ordenada, na qual cada item da lista é marcado com um símbolo, geralmente um ponto. Os itens da lista também são definidos pela tag

  • .

    li

    (Voltar para o Índice)

    (List Item):Essa tag é usada para definir um item em uma lista (ol ou ul). Cada item deve ser colocado dentro dessa tag.

    a

    (Voltar para o Índice)

    (Anchor):Essa tag é usada para criar um link ou âncora em um documento HTML. É usada em conjunto com o atributo href para especificar o URL de destino do link.

    hr

    (Voltar para o Índice)

    (Horizontal Rule):Essa tag é usada para criar uma linha horizontal que divide seções de conteúdo em uma página. Não requer uma tag de fechamento e não possui conteúdo entre as tags.

    sub

    (Voltar para o Índice)

    (Subscript):Essa tag é usada para exibir o texto em formato de subscrito, posicionando-o ligeiramente abaixo da linha de base normal.

    sup

    (Voltar para o Índice)

    (Superscript):Essa tag é usada para exibir o texto em formato de sobrescrito, posicionando-o ligeiramente acima da linha de base normal.

    blockquote

    (Voltar para o Índice)

    (Block Quote):Essa tag é usada para citar um trecho de texto que é destacado do restante do conteúdo. O texto citado geralmente é recuado e estilizado de maneira diferente para distingui-lo.

    font

    (Voltar para o Índice)

    (Font):Essa tag é usada para definir a fonte, tamanho, cor e outros estilos de texto. No entanto, essa tag não é mais recomendada, e é preferível usar CSS para estilizar o texto.

    del

    (Voltar para o Índice)

    (Deleted Text):Essa tag é usada para indicar que um trecho de texto foi removido ou deletado. O texto dentro dessa tag geralmente é exibido com uma linha horizontal atravessando-o.

    abbr

    (Voltar para o Índice)

    (Abbreviation):Essa tag é usada para marcar uma abreviação ou sigla no texto. Geralmente é exibido como texto com uma linha pontilhada que, quando o cursor é passado sobre ele, mostra o significado completo da abreviação.

    Outras tags citadas no curso

    • form (Form):Essa tag é usada para criar um formulário em uma página HTML. Ela envolve os elementos do formulário, como campos de entrada, botões e áreas de texto.
    • input (Input):Essa tag é usada para criar um campo de entrada em um formulário. Existem vários tipos de campos de entrada, como texto, senha, número, data, e-mail, entre outros, que são especificados pelo atributo type.
    • checkbox> (Checkbox):Essa tag é usada para criar uma caixa de seleção em um formulário. Os usuários podem marcar ou desmarcar uma caixa de seleção para indicar uma escolha ou opção.
    • radio (Radio Button):Essa tag é usada para criar um botão de opção em um formulário. Os botões de opção são mutuamente exclusivos, permitindo que o usuário selecione apenas uma opção dentro de um grupo de botões relacionados.
    • button (Button):Essa tag é usada para criar um botão em uma página HTML. Os botões podem ser usados para enviar formulários, executar ações ou interagir com o usuário de outras maneiras.
    • select (Select):Essa tag é usada para criar uma lista suspensa em um formulário. Os usuários podem selecionar uma opção em uma lista de opções fornecidas.
    • textarea (Text Area):Essa tag é usada para criar uma área de texto de várias linhas em um formulário. Os usuários podem inserir texto em várias linhas nessa área.
    • img (Image):Essa tag é usada para inserir uma imagem em uma página HTML. O atributo src especifica o URL da imagem e o atributo alt fornece um texto alternativo para casos em que a imagem não pode ser exibida.
    • audio (Audio):Essa tag é usada para incorporar áudio em uma página HTML. O atributo src especifica o URL do arquivo de áudio e a tag pode conter elementos adicionais, como controles de reprodução.
    • video (Video):Essa tag é usada para incorporar vídeos em uma página HTML. O atributo src especifica o URL do arquivo de vídeo e a tag pode conter elementos adicionais, como controles de reprodução.
    • track (Track):Essa tag é usada para fornecer legendas, descrições de áudio ou outras faixas de texto para elementos audio ou video. É usado em conjunto com os atributos src e kind para especificar o arquivo e o tipo de faixa.
    • iframe (Inline Frame):Essa tag é usada para incorporar uma página web dentro de outra página HTML. O atributo src especifica o URL da página a ser incorporada.
    • table (Table):Essa tag é usada para criar uma tabela em uma página HTML. Ela é composta por linhas (tr) e células (td ou th).
    • tr (Table Row):Essa tag é usada para criar uma linha em uma tabela. Ela contém células de tabela (td ou th).
    • td (Table Data Cell):Essa tag é usada para criar uma célula de dados em uma tabela. Ela está contida dentro de uma tr e contém o conteúdo da célula.
    • th (Table Header Cell):Essa tag é usada para criar uma célula de cabeçalho em uma tabela. Ela está contida dentro de uma tr e contém o conteúdo do cabeçalho da coluna.
    • tbody (Table Body):Essa tag é usada para agrupar as linhas de uma tabela que contêm dados. Geralmente é usado em conjunto com as tags tr.
    • thead (Table Head):Essa tag é usada para agrupar as linhas de cabeçalho de uma tabela. Geralmente é usado em conjunto com as tags tr.
    • tfoot (Table Foot):Essa tag é usada para agrupar as linhas de rodapé de uma tabela. Geralmente é usado em conjunto com as tags tr.
    • main (Main Content):Essa tag é usada para marcar o conteúdo principal de uma página HTML. Geralmente é usada apenas uma vez por página.
    • header (Header):Essa tag é usada para criar um cabeçalho em uma página HTML. Geralmente contém elementos de identificação, logotipos, menus de navegação ou outros conteúdos de cabeçalho.
    • footer (Footer):Essa tag é usada para criar um rodapé em uma página HTML. Geralmente contém informações de direitos autorais, links de contato, informações de autoria ou outros conteúdos de rodapé.
    • aside (Aside):Essa tag é usada para criar um conteúdo relacionado ou auxiliar em uma página HTML. Geralmente é usado para conteúdo secundário, como barras laterais, anúncios, notas de rodapé ou informações relacionadas.
    • section (Section):Essa tag é usada para agrupar conteúdo relacionado em uma página HTML. É uma maneira de organizar e estruturar o conteúdo, semelhante a um contêiner.
    • nav (Navigation):Essa tag é usada para criar um menu de navegação em uma página HTML. Geralmente contém links para diferentes seções ou páginas do site.
    • article (Article):Essa tag é usada para marcar um conteúdo independente e autossuficiente em uma página HTML. É usado para artigos de notícias, postagens de blog, comentários ou outros tipos de conteúdo autônomo.
    • q (Inline Quotation):Essa tag é usada para marcar uma citação curta em linha dentro de um parágrafo. O texto dentro dessa tag geralmente é exibido com aspas.
    • figure (Figure):Essa tag é usada para marcar uma figura, como uma imagem, diagrama, ilustração ou gráfico, em uma página HTML. Pode ser usado em conjunto com figcaption para adicionar uma legenda à figura.
    • figcaption (Figure Caption):Essa tag é usada para adicionar uma legenda a uma figura marcada com a tag figure. A legenda fornece uma descrição ou explicação para a figura.
    • picture (Picture):Essa tag é usada para fornecer diferentes versões de uma imagem para diferentes dispositivos ou condições de exibição. Ela pode conter várias tags source para especificar diferentes fontes de imagem.
    • datalist (Data List):Essa tag é usada para fornecer uma lista de opções predefinidas para um campo de entrada (input do tipo text ou search). Os usuários podem selecionar uma opção da lista suspensa ao digitar.
    • code (Code):Essa tag é usada para marcar uma seção de código em uma página HTML. O texto dentro dessa tag geralmente é exibido com uma fonte monoespaçada para distinguir do restante do texto.
    • kbd (Keyboard Input):Essa tag é usada para marcar uma entrada de teclado em uma página HTML. O texto dentro dessa tag geralmente é exibido em negrito e em uma fonte monoespaçada.
    • pre (Preformatted Text):Essa tag é usada para preservar a formatação de um bloco de texto pré-formatado em uma página HTML. O texto dentro dessa tag é exibido exatamente como foi digitado, incluindo espaços em branco e quebras de linha.
    • details (Details):Essa tag é usada para criar um elemento com detalhes ocultos que podem ser revelados ou ocultados pelo usuário. É geralmente usado em conjunto com a tag summary.
    • summary (Summary):Essa tag é usada para fornecer um resumo ou título para um elemento details. Geralmente é exibido como um cabeçalho clicável para revelar ou ocultar os detalhes.
    • meter (Meter):Essa tag é usada para criar uma barra de medição ou indicador em uma página HTML. É usado para representar um valor numérico dentro de um intervalo específico.
    • progress (Progress):Essa tag é usada para criar uma barra de progresso em uma página HTML. Ela representa o progresso de uma tarefa ou operação.
    • canvas (Canvas):Essa tag é usada para criar uma área de desenho em uma página HTML. Ela fornece uma superfície na qual é possível desenhar gráficos, animações, gráficos vetoriais e outros elementos visuais.
    • Address em HTML é usada para fornecer informações de contato ou detalhes de localização dentro de um documento. Ela é usada para indicar que o texto ou conteúdo dentro da tag representa informações de contato ou endereço.
